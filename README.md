# 닥터마틴 제품 데이터 크롤링 및 소비자 리뷰 분석 프로젝트

이 프로젝트는 '닥터마틴(Dr. Martens)' 공식 홈페이지의 제품 데이터와 소비자 리뷰를 수집하고, 이를 시각화 및 텍스트 마이닝 기법으로 분석하여 인사이트를 도출하는 과정을 담고 있습니다.

## 📂 파일 구성 및 설명

### 1. `크롤링.ipynb`
- **목적**: 닥터마틴 공식 홈페이지의 베스트셀러 제품 정보 수집
- **주요 기능**:
    - **Selenium**을 활용한 웹 브라우저 자동화 제어
    - 남성/여성 카테고리별 제품명, 색상, 가격, 리뷰 수, 별점 데이터 추출
    - 추출된 데이터를 **Pandas DataFrame**으로 변환 및 성별(gender) 컬럼 생성 등 전처리
    - 최종 데이터를 `닥터마틴_best_df.csv` 파일로 저장

### 2. `1461 모노 리뷰 크롤링.ipynb`
- **목적**: 특정 인기 모델인 '1461 모노' 제품의 상세 리뷰 텍스트 집중 수집
- **주요 기능**:
    - 특정 제품 페이지에 접속하여 실사용자들의 리뷰 텍스트를 대량으로 크롤링
    - **tqdm** 라이브러리를 사용하여 크롤링 진행 상황을 시각적으로 표시
    - 텍스트 분석(텍스트 마이닝)을 위한 기초 원천 데이터 확보

### 3. `시각화.ipynb`
- **목적**: 수집된 제품 데이터를 통계적으로 분석하고 시각화
- **주요 기능**:
    - **Matplotlib** 및 **Seaborn** 라이브러리를 활용한 데이터 시각화
    - 가격(price), 리뷰 수(review_cnt), 별점(rating) 등 수치형 데이터의 기초 통계량 파악
    - 박스 플롯(Box Plot)을 활용한 이상치(Outlier) 탐지 및 데이터 분포 확인
    - 한글 폰트(맑은 고딕 등) 설정으로 그래프 내 한글 깨짐 방지 처리

### 4. `1461_모노 텍스트마이닝.ipynb`
- **목적**: 크롤링한 리뷰 데이터를 바탕으로 소비자 반응 및 감성 분석 수행
- **주요 기능**:
    - **텍스트 전처리**: 가상 리뷰 생성 및 실제 수집된 리뷰 텍스트 통합
    - **빈도 분석**: 리뷰에서 자주 등장하는 핵심 키워드 추출
    - **워드 클라우드(Word Cloud)**: 주요 키워드를 시각화하여 제품의 강점 및 약점 파악
    - **만족도 수치화**: 단순 사전 방식 및 가중치 방식을 활용한 제품 만족도 점수 계산
    - **LDA 토픽 모델링**: (코드 포함 시) 리뷰 내 주요 주제군을 분류하는 기법 적용 시도

## 🛠 기술 스택
- **Language**: Python
- **Libraries**:
    - 데이터 수집: `Selenium`, `chromedriver_autoinstaller`, `BeautifulSoup`
    - 데이터 분석: `Pandas`, `Numpy`
    - 데이터 시각화: `Matplotlib`, `Seaborn`, `WordCloud`
    - 자연어 처리: `KoNLPy` (또는 관련 형태소 분석기)
